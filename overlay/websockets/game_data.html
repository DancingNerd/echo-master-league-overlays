Game Data Endpoints

These endpoints are generally json responses with information about the current game state.

  • Home
  • Game Stats

  • /session

    A direct re-host of the EchoVR API, but without CORS restrictions.

  • /stats

    JSON for the current stats collected by Spark. Includes VRML team names and logos from the VRML API based on the player list.

  • Spark Websockets API

    Subscribe to events in the browser. This makes event-driven overlays very simple.
